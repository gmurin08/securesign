## About SecureSign

SecureSign provides secure, easy-to-use document signing capabilities. Built on the robust foundation of Documenso, SecureSign inherits all the powerful features of the open source signing infrastructure while being customized for Alder Creek Digital's needs.

## Features

- Digital document signing
- Multi-recipient support
- Customizable signing workflows
- Email notifications
- Secure document storage
- Audit trails and compliance

## Tech Stack

- [TypeScript](https://www.typescriptlang.org/) - Language
- [React Router](https://reactrouter.com/) - Framework
- [Prisma](https://www.prisma.io/) - ORM
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [shadcn/ui](https://ui.shadcn.com/) - Component Library
- [react-email](https://react.email/) - Email Templates
- [tRPC](https://trpc.io/) - API

## Local Development

### Requirements

- Node.js (v22 or above)
- PostgreSQL Database
- Docker (optional)

### Developer Quickstart

1. Clone the repository
2. Copy `.env.example` to `.env` and configure your environment variables
3. Run `npm run dx` to start the database and mail server
4. Run `npm run dev` to start the development server
5. Access the application at http://localhost:3000

For more detailed setup instructions, please refer to the [Documenso documentation](https://github.com/documenso/documenso).
